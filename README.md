# Janayojana - Government Schemes Aggregator
Janayojana is a user-friendly website that simplifies the process of finding and understanding various Indian government schemes. It helps citizens easily access detailed information on available schemes through intuitive search and filtering options, making it easier to identify opportunities and benefits they are eligible for.  

# 🚀 Live Demo  
[Visit the Website](https://janayojana.netlify.app)  
Janayojana by Team CodeBuilders

# 📌 Features  
1. Detailed Scheme Information  
   + Clear, concise details on each scheme, including eligibility criteria, application steps, deadlines, and benefits.  

2. Search and Filter Options  
   + Category-Based Search: Browse schemes by categories such as education, healthcare, employment, etc.
   + State-Based Selection: Filter schemes based on specific states to find region-specific benefits.

3. Regional Language Support
   + Integrated with Google Translate API to provide multilingual support for users across India.

4. Application Assistance
   + Step-by-step guides and FAQs to help navigate the application process.
   + Direct links to official portals for submitting applications.

5. Regular Updates
   + Timely updates on new schemes, changes to existing ones, and upcoming deadlines to ensure users never miss out on opportunities.

6. Secure Data Handling
   + Focus on user data privacy and secure encryption for safe browsing.
